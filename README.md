# Portfolio

Bold, early‑2000s tech‑ad inspired portfolio built as a single HTML file with custom animations, interactive swirl controls, and a rolling skills stack.

## Highlights
- One‑page layout with themed sections and scroll‑reveal motion
- Interactive swirl tiles + controls
- Responsive layout with mobile‑tuned typography and spacing
- Custom cursor and scanline texture for retro vibe

## Tech Stack
- HTML, CSS, JavaScript (no frameworks)
- Google Fonts: Bebas Neue, Space Mono

## Structure
- Main file: [index.html](index.html)
- Local font asset: [bebasneue.woff2](bebasneue.woff2)

## Run Locally
- Open [index.html](index.html) in your browser
